# go-project-template

A GitHub repository template to be used as a starting point for projects written in Go.

## Usage

The following files require your attention immediately after creating a
repository from this template:

- [ ] `.github/CODEOWNERS`
- [ ] `.github/workflows/lint.yml`
- [ ] `.github/workflows/release.yml`
- [ ] `cmd/app/main.go`
- [ ] `go.mod`
- [ ] `Makefile`
- [ ] `README.md`
